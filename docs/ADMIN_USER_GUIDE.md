> เข้าใช้งาน: `https://admin.re-ya.com` (หรือ `http://localhost:3001`)

---

## เข้าสู่ระบบ

1. เปิดเว็บ Admin Dashboard
2. กรอก Email และ Password ที่ได้รับจากผู้ดูแลระบบ
3. กด "เข้าสู่ระบบ"

ระบบจะจำ session ไว้ 7 วัน ถ้า token หมดอายุจะ refresh อัตโนมัติ

## คิวใบสั่งยา

### การตรวจสอบใบสั่งยา
1. เข้าเมนู "คิวใบสั่งยา"
2. ใบสั่งยาเรียงตาม priority (STAT > URGENT > ROUTINE) และ SLA
3. กดที่ใบสั่งยาเพื่อดูรายละเอียด
4. ตรวจสอบ: ภาพใบสั่งยา, ข้อมูลที่ OCR อ่านได้, ประวัติผู้ป่วย
5. ดู AI Safety Alerts (drug interaction, allergy, contraindication)
6. กด "อนุมัติ" หรือ "ปฏิเสธ" พร้อมบันทึกเหตุผล

### SLA
- STAT: ต้องตรวจภายใน 15 นาที
- URGENT: ต้องตรวจภายใน 30 นาที
- ROUTINE: ต้องตรวจภายใน 60 นาที

## จัดการออเดอร์

### ดูออเดอร์ทั้งหมด
1. เข้าเมนู "ออเดอร์"
2. กรองตามสถานะ: รอชำระ, ชำระแล้ว, กำลังจัด, แพ็คแล้ว, จัดส่งแล้ว

### ตรวจสลิปการชำระเงิน
1. กดปุ่ม "ตรวจสลิป" ที่มุมขวาบน
2. ดูรายการออเดอร์ที่รอตรวจสลิป
3. กด "ดูสลิป" เพื่อดูรูปสลิป
4. กด "อนุมัติ" หรือ "ปฏิเสธ"

### คืนเงิน
1. เข้าหน้ารายละเอียดออเดอร์
2. กดปุ่ม "คืนเงิน"
3. ระบุเหตุผล
4. ยืนยัน

### เปลี่ยนสถานะออเดอร์
1. เข้าหน้ารายละเอียดออเดอร์
2. กดปุ่มดำเนินการตามลำดับ:
   - ชำระแล้ว → "เริ่มจัดยา"
   - กำลังจัด → "แพ็คเสร็จ"
   - แพ็คแล้ว → "จัดส่ง"
   - จัดส่งแล้ว → "ส่งถึงแล้ว"

## ข้อมูลผู้ป่วย

### ค้นหาผู้ป่วย
1. เข้าเมนู "ผู้ป่วย"
2. พิมพ์ชื่อ, เบอร์โทร, หรือ HN ในช่องค้นหา

### ดูรายละเอียดผู้ป่วย
- ข้อมูลส่วนตัว (อายุ, เพศ, น้ำหนัก, กรุ๊ปเลือด)
- ข้อมูลติดต่อ (โทรศัพท์, อีเมล, LINE)
- แพ้ยา (ชื่อยา, ความรุนแรง, อาการ)
- โรคประจำตัว (ชื่อโรค, ICD-10, สถานะ)
- ยาที่ใช้อยู่ (ชื่อยา, ขนาด, วิธีใช้)
- ประวัติใบสั่งยา
- Adherence Stats (อัตราทานยา, streak)
- Loyalty Points (แต้ม, tier, ปรับแต้ม)

## ADR Reports

### ดูรายงาน ADR
1. เข้าเมนู "ADR Reports"
2. กรองตามความรุนแรง (Mild/Moderate/Severe/Life-threatening)
3. กรองตามสถานะ (Reported/Under Review/Assessed/Closed)

### ประเมิน Causality
1. กดที่รายงาน ADR เพื่อดูรายละเอียด
2. เลือก Causality (WHO-UMC): Certain/Probable/Possible/Unlikely
3. เพิ่มหมายเหตุ
4. กด "บันทึกการประเมิน"

### Export สำหรับ อย.
1. กดปุ่ม "Export อย." ที่มุมขวาบน
2. ระบบจะสร้างไฟล์ตามรูปแบบ อย.

## Clinical Services

### Medication Review (MR)
1. เข้าเมนู "Clinical" → tab "Medication Review"
2. ดูคิว MR ที่รอดำเนินการ
3. กด "ดำเนินการ" เพื่อเข้าหน้า review
4. กรอก Findings และ Recommendations
5. กด "บันทึก"

### TDM (Therapeutic Drug Monitoring)
1. เข้าเมนู "Clinical" → tab "TDM"
2. ดูคิว TDM ที่รอบันทึกผล
3. กด "บันทึกผล"
4. กรอกระดับยาที่วัดได้, หน่วย, คำแนะนำ
5. กด "บันทึกผล"

---

## Telemedicine

### รับเรื่องปรึกษา
1. เข้าเมนู "Telemedicine"
2. ดูคิวที่รอดำเนินการ
3. กดที่รายการเพื่อดูรายละเอียด
4. กด "รับเรื่อง" เพื่อรับ consultation
5. ระบบจะสร้าง Agora channel สำหรับ video call

### KYC Review
1. เข้าเมนู "KYC Review"
2. ค้นหาด้วย Patient ID
3. ดูสถานะ KYC และข้อมูลที่ตรวจสอบ
4. กด "อนุมัติ" หรือ "ปฏิเสธ" พร้อมหมายเหตุ

## ตั้งค่า (Super Admin)

### ตั้งค่าทั่วไป
- ชื่อร้าน, โทรศัพท์, อีเมล, ที่อยู่, เวลาเปิด-ปิด
- บันทึกลง database อัตโนมัติ

### เพิ่ม Staff
1. เข้า tab "Staff"
2. กรอก Email, Password, ชื่อ, นามสกุล, Role
3. กด "เพิ่ม Staff"

### การแจ้งเตือน
- เปิด/ปิดการแจ้งเตือน: ออเดอร์ใหม่, สต็อกต่ำ, ใบสั่งยารอตรวจ

### ระบบ
- ดูสถานะ API, PostgreSQL, Redis, Meilisearch, MinIO
- Sync Products จาก Odoo
- Check Odoo connection
