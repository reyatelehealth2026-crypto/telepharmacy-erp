# ข้อตกลงและยินยอมการรับบริการเภสัชกรรมทางไกล

**เวอร์ชัน 1.0.0**  
**มีผลตั้งแต่: 1 มกราคม 2568**

---

## 1. ข้อจำกัดของเทคโนโลยี

ข้าพเจ้าเข้าใจและยอมรับว่า การรับบริการเภสัชกรรมทางไกล (Telepharmacy) มีข้อจำกัดทางเทคโนโลยีดังนี้:

- **เภสัชกรไม่สามารถตรวจร่างกายโดยตรงได้** - การประเมินอาการจะอาศัยเพียงการสื่อสารผ่านภาพและเสียงเท่านั้น
- **คุณภาพของภาพและเสียงขึ้นอยู่กับสัญญาณอินเทอร์เน็ต** - สัญญาณไม่ดีอาจทำให้การสื่อสารไม่ชัดเจน
- **อาจมีความล่าช้าในการสื่อสาร** - เนื่องจากข้อจำกัดของเครือข่าย
- **ไม่สามารถทำการตรวจทางห้องปฏิบัติการได้** - ไม่สามารถตรวจเลือด ตรวจปัสสาวะ หรือตรวจอื่นๆ ได้

ข้อจำกัดเหล่านี้อาจส่งผลต่อความแม่นยำในการประเมินอาการและการให้คำแนะนำ

## 2. ขอบเขตการให้บริการ

ข้าพเจ้าเข้าใจว่าบริการเภสัชกรรมทางไกล **เหมาะสำหรับ**:

✅ การติดตามอาการโรคเรื้อรังที่มีประวัติการรักษาแล้ว (เช่น เบาหวาน ความดันโลหิตสูง)  
✅ การขอยาประจำตัวเดิมที่เคยใช้มาแล้ว  
✅ การปรึกษาเรื่องยาและการใช้ยา  
✅ อาการเจ็บป่วยเล็กน้อยที่ไม่ซับซ้อน (เช่น ไข้หวัด ปวดศีรษะเล็กน้อย)

และ **ไม่เหมาะสำหรับ**:

❌ อาการเฉียบพลันรุนแรง (ปวดท้องมาก เจ็บหน้าอก หอบเหนื่อย ไข้สูงเกิน 39°C)  
❌ ผู้ป่วยใหม่ที่ไม่มีประวัติการรักษา  
❌ การขอยาเสพติดให้โทษหรือยาควบคุมพิเศษ  
❌ กรณีที่ต้องตรวจร่างกายหรือตรวจทางห้องปฏิบัติการ

## 3. หน้าที่ในการปฏิบัติตามคำแนะนำ

ข้าพเจ้ายินยอมที่จะ:

1. **ให้ข้อมูลที่ถูกต้องและครบถ้วนแก่เภสัชกร** - รวมถึงประวัติการแพ้ยา โรคประจำตัว และยาที่ใช้อยู่
2. **ปฏิบัติตามคำแนะนำของเภสัชกรอย่างเคร่งครัด** - ใช้ยาตามที่แนะนำ ไม่เพิ่มหรือลดขนาดยาเอง
3. **ไปพบแพทย์หรือไปโรงพยาบาลทันที** หากเภสัชกรแนะนำให้ส่งต่อ

⚠️ **ข้อสำคัญ**: ข้าพเจ้าเข้าใจว่า หากข้าพเจ้าเพิกเฉยต่อคำแนะนำให้ไปโรงพยาบาล ข้าพเจ้ายินดีรับผิดชอบผลลัพธ์ทางสุขภาพที่เกิดขึ้นด้วยตนเอง และร้านยาไม่มีความรับผิดชอบใดๆ

## 4. ความเป็นส่วนตัวและการบันทึกข้อมูล

ข้าพเจ้ายินยอมให้:

- **บันทึกภาพและเสียงการให้คำปรึกษาทั้งหมด** - เพื่อความปลอดภัยและการตรวจสอบคุณภาพ
- **เก็บรักษาข้อมูลไว้เป็นเวลาอย่างน้อย 10 ปี** ตามกฎหมายว่าด้วยการประกอบโรคศิลปะ
- **ใช้ข้อมูลเพื่อการรักษา การตรวจสอบคุณภาพ และการป้องกันข้อพิพาททางกฎหมาย**

🔒 ข้อมูลทั้งหมดจะถูกเก็บรักษาอย่างปลอดภัยและเป็นความลับตาม **พระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA)**

ข้อมูลจะถูกเข้ารหัสและจัดเก็บในประเทศไทยเท่านั้น ไม่มีการส่งข้อมูลออกนอกประเทศ

## 5. การยกเลิกความยินยอม

ข้าพเจ้าสามารถถอนความยินยอมนี้ได้ตลอดเวลา โดยการแจ้งเป็นลายลักษณ์อักษรผ่านระบบหรือติดต่อร้านยาโดยตรง

ทั้งนี้ การถอนความยินยอมจะมีผลดังนี้:
- มีผลตั้งแต่วันที่แจ้ง (ใช้เวลาดำเนินการ 7 วันทำการ)
- ไม่ส่งผลย้อนหลังต่อการให้บริการที่ได้รับไปแล้ว
- ข้อมูลที่จำเป็นต้องเก็บตามกฎหมายจะยังคงถูกเก็บไว้ตามระยะเวลาที่กฎหมายกำหนด

## 6. สิทธิของผู้ป่วยตาม PDPA

ข้าพเจ้ามีสิทธิ์:

- **เข้าถึงและขอสำเนาข้อมูลส่วนบุคคล** ของตนเอง
- **ขอให้แก้ไขข้อมูล** ที่ไม่ถูกต้องหรือไม่สมบูรณ์
- **ขอให้ลบหรือทำลายข้อมูล** (ยกเว้นข้อมูลที่กฎหมายกำหนดให้เก็บไว้)
- **ขอให้ระงับการใช้ข้อมูล** ในบางกรณี
- **คัดค้านการประมวลผลข้อมูล** ในบางกรณี
- **ขอให้โอนย้ายข้อมูล** ในรูปแบบที่อ่านได้ด้วยเครื่องจักร

สามารถติดต่อเจ้าหน้าที่คุ้มครองข้อมูลส่วนบุคคลได้ที่: privacy@telepharmacy.com
